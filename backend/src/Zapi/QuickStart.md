# Quick Start

n.b. This is a fastAPI app

## How to run the server

First activate relevant pyenv (zasshi) then run
```
#pyenv activate zasshi

poetry shell
```

### Option v1:
```
uvicorn main:app --reload
```

### Option v2:
```
uvicorn main_api:app --reload
```

## Check it

use postman to submit a query
